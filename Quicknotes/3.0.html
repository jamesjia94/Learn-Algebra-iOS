Terms

3.1

Linear System: two equations that use the same variables

Substitution Method: a method for solving linear systems by solving for one variable, then substituting this solution into the original system

3.2

Elimination Method: a method for solving linear systems by arranging the equations to “eliminate” one variable, then solve for the second variable

3.3

Graphing Method: a method for solving linear systems by graphing the two equations and looking for the interception point

Sample Solutions

3.1

Substitution:

\(x-3y=2\)

\(2x+2y=12\)

Solve for x: \(x=3y+2\)

Substitute into the other original equation: \(2\left(3y+2\right)+2y=12\)

Simplify: \(6y + 4 + 2y = 12 \rightarrow 8y = 8 \rightarrow y = 1\)

Now that we have y, solve for x with either equation: \(x - 3\left(1\right)=2 \rightarrow x - 3 = 2 \rightarrow x = 5\)

Or: \(2x+2\left(1\right)=12 \rightarrow 2x + 2 = 12 \rightarrow 2x = 10 \rightarrow x = 5\)

Answer: \(x = 5, y = 1\)

3.2

Elimination:

\(2x + y = 10\)

\(2x - 3y = 2\)

Eliminate one variable by subtracting the two equations from both: \(\left(2x - 2x\right)+\left(y-\left(-3y\right)\right)=\left(10 - 2\right)\)

Simplify: \(4y = 8 \rightarrow y = 2\)

Solve for the second variable by substituting in this solution into either equation: \(2x + 2 = 10 \rightarrow 2x = 8 \rightarrow x = 4\)

Or: \(2x-3\left(2\right)=2 \rightarrow 2x - 6 = 2 \rightarrow 2x = 8 \rightarrow x = 4\)

Answer: \(x=4, y=2\)

Elimination With Different Coefficients

\(3x+2y=11\)

\(2x-y=5\)

Multiply/divide the equations by constants to create equal coefficients on one variable; for this example, we will multiply the second equation by 2: \(2\left(2x-y\right)=2\left(5\right) \rightarrow 4x - 2y = 10\)

Use this new equation with the original first equation:

\(3x+2y=11\)

\(4x-2y=10\)

Add the two equations to eliminate y: \(\left(3x + 4x\right)+\left(2y + \left(-2y\right)\right)=\left(11+10\right)\)

Simplify: \(7x=21 \rightarrow x=3\)

Solve for second variable with either equation:

\(3\left(3\right)+2y=11 \rightarrow 9 + 2y = 11 \rightarrow 2y = 2 \rightarrow y = 1\)

Or: \(2\left(3\right)-y=5 \rightarrow 6 - y = 5 \rightarrow y = 1 \)

Answer: \(x=3, y=1\)
