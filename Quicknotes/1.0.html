Terms

1.1

Variable: a letter that is used to represent a number

Expression: a phrase that is used to represent a number

Equation: expressions that are set equal to each other with an equal sign

Algebraic Expression: an expression with at least one variable

Term: parts of an expression

1.2

Solution: a value that replaces a variable to make the equation true

Reciprocal: the inverse of the original fraction; for example, the reciprocal of \(\frac{4}{5}\) is \(\frac{5}{4}\)

Contradictions: equations that cannot have a solution because of a logical error

Identity: an equation that is always true no matter the value of the variable(s)

1.3

Inequalities: equations that are related together that may not (and sometimes cannot) be equal to each other

\(a > b\): a is greater than b

\(a < b\): a is less than b

\(a \ge b\): a is greater than or equal to b

\(a \le b\): a is less than or equal to b

1.4

Ratio: a relationship between two numbers

Rates: ratios comparing different units; for example, miles/hour

Proportions: an equation that sets two ratios equal to each other

1.5

Exponents: a mathematical operation that, in the form \(x^n\), multiplies a number x by itself n times

1.7

Scientific Notation: a way of expressing a number (often very large or very small) in the form \(a*10^n\); for example, 50000 would be \(5.00*10^4\)

Formulas


1.1

Additive Identity: \(x+0=x\)

Multiplicative Identity: \(x*1=x\)

Combining Terms: \(ax + bx = (a + b)x\)

Associative Property: \( \left(a+b\right)+c=a+\left(b+c\right)\)

Commutative Property: \(a+b=b+a\)

Distributive Property: \( a\left(b+c\right)=ab + ac\)

1.2

Addition Property of Equality: If \(a = b\), then \(a + c = b + c\)

Subtraction Property of Equality: If \(a = b\), then \(a - c = b - c\)

Multiplication Property of Equality: If \(a = b\), then \(a * c = b * c\)

Division Property of Equality: If \(a = b\), then \(\frac{a}{c}=\frac{b}{c}\) as long as \(c \ne 0 \)

1.4

Cross-Multiplication: \(\frac{a}{b}=\frac{c}{d}\rightarrow ad = bc\)

1.5

Product Rule of Exponents: \(x^n*x^m=x^{n+m}\)

Quotient Rule of Exponents: \(\frac{x^n}{x^m}=x^{n-m}\)

Power Rule of Exponents: \({\left(x^n\right)}^m=x^{n*m}\)

Power Rule for a Product: \({\left(xy\right)}^n=x^ny^n\)

1.6

Negative Exponents: \(x^{-n}=\frac{1}{x^n}\)

1.7

Multiplying Numbers in Scientific Notation: \(\left(a*10^n\right)*\left(b*10^m\right)=\left(a*b\right)*10^{n+m}\)

Dividing Numbers in Scientific Notation: \(\frac{a*10^n}{b*10^m}=\left(\frac{a}{b}\right)*10^{n-m}\)
