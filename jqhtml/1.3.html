Inequalities

In the past, we have discussed equations, which set two expressions equal to each other. Now, we will cover inequalities, which relate the values of two expressions that may not (and often cannot) be equal to each other. Here are the main signs used for inequalities:

Sign Definition
$a > b$ $a$ is greater than $b$
$a < b$ $a$ is less than $b$
$a ≥ b$ $a$ is greater than or equal to $b$
$a ≤ b$ $a$ is less than or equal to $b$

Algebraic inequalities relate the values of two expressions using these signs instead of an equals sign like in an equation. For example, $x=2$ is an equation, and $x<2$ is an inequality. This inequality states that $x$ is less than $2$. While an equation would have a set number of solutions, usually one, an inequality has the potential for an infinite number of solutions. For the above example, there are an infinite number of values that are less than $2$.
For the most part, inequalities can be solved in the same way you solve equations. For example, take the inequality $3x + 7 < 19$. We can subtract $7$ from both sides, simplifying the inequality to $3x < 12$. Dividing both sides by $3$ gives us $x < 4$. At this point, we can stop; this is the solution.
One thing to note when solving inequalities is that, when multiplying or dividing with a negative number, you need to reverse the inequality sign. If we were to solve the inequality $-2x > 20$, we would divide both sides by $-2$ to remove the coefficient from $x$. Because we divided by a negative number, we must switch the greater than sign to the less than sign. The solution would then be $x < -10$. Below is a table of each sign and its opposite.

Sign Opposite of the Sign
$>$ $<$
$<$ $>$
$≤$ $≥$
$≥$ $≤$
